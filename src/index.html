Add view-markup.js before any script tags at the bottom of the body. Add the data-view-markup attribute to the any html element inside the body that you want to view the markup for.

View Markup setup and options

Add the below options inside the data-view-markup value.

  • Add title to modal
    Example:
    data-view-markup="title: Block Name"
  • Adjust markup modal button x position
    Example:
    data-view-markup="btn-x: 10px"
  • Adjust markup modal button y position
    Example:
    data-view-markup="btn-y: 10px"
  • Adjust markup modal button z-index
    Example:
    data-view-markup="btn-z: 2"
  • Adjust markup modal button position (might need this when placing the button in a custom spot)
    Example:
    data-view-markup="btn-pos: relative"
  • Display a snippet of javascript that goes along with html
    Example:
    data-view-markup="associated-script: #scriptTagID"
  • Customize backdrop color (rgb values)
    Example:
    data-view-markup="backdrop-rgb: 255, 255, 255"
  • Render in page (after element)
    Example:
    data-view-markup="render-in-page: true"
  • Multiple options (separate with a semi-colon)
    Example:
    data-view-markup="title: Block Name; btn-x: -3px; btn-z: 3"

Tablist setup and options

Wrap view markup items inside and element with the data-view-markup-tabs attribute.

Options below for customization

  • Add title to tabs navigation
    Example:
    data-view-markup-tabs="title: Tab section name"
  • Add max-with to tabs navigation
    Example:
    data-view-markup-tabs="max-width: 1200px"
  • Adjust horizontal outer padding on the tabs navigation
    Example:
    data-view-markup-tabs="padding-x: 28px"
  • Add class(es) to tabs navigation
    Example:
    data-view-markup-tabs="class: container px-5"
  • Remove background color
    Example:
    data-view-markup-tabs="background: false"
  • Pill styled navigation tab items
    Example:
    data-view-markup-tabs="pill-view: true"
  • Show modal button in tab nav bar
    Example:
    data-view-markup-tabs="modal-btn: true"

Query param settings

Add the below settings as a query param to the view-markup script tag src.

  • modal-nav - Activate navigation arrows in modal to be able to navigate between view-markup elements
    Example:
    js/view-markup.min.js?modal-nav
  • dynamic-pos - Dynamically position view markup modal button(s) without modifying the DOM around the view-markup element
    Example:
    "js/view-markup.min.js?dynamic-pos"
  • exclude-attribute
    Example:
    "js/view-markup.min.js?exclude-attribute=data-jtn-anchor,example-attr"
  • Multiple params
    Example:
    "js/view-markup.min.js?dynamic-pos&modal-nav"
